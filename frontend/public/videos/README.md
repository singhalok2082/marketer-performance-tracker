# Landing page videos

Drop these five files here and the landing page will pick them up automatically —
no code changes needed. Until they exist, each section just shows its dark
liquid-glass background with no video (a 404 on a `<video>` source fails silently).

| File            | Used in                         | Suggested shape        |
|-----------------|----------------------------------|-------------------------|
| `hero.mp4`       | Full-screen hero background      | Vertical-ish / tall, object-cover crops to fill |
| `featured.mp4`   | "Our Approach" featured panel     | 16:9, loops             |
| `tracking.mp4`   | "Tracking × Visibility" left panel| 4:3, loops              |
| `service-1.mp4`  | Services card 1 ("Research & Insight") | 16:9, loops        |
| `service-2.mp4`  | Services card 2 ("Design & Execution") | 16:9, loops        |

Screen recordings of the actual dashboard in use (someone logging a profile,
scrolling the team view, etc.) work well here — they double as real product
demos instead of stock footage. Keep clips short (5-15s) and silent/muted-safe
since all of these autoplay muted.
